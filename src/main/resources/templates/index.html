Práctica 4

Información

Esta práctica cuenta con una base de datos en la que se encuentran libros, editoriales y autores

Acceso a la consola

Mediante la consola H2 podemos acceder a las distintas tablas almacenadas en la base de datos y realizar consultas y sentencias SQL.

Para acceder a la consola de la base de datos, hay que acceder a H2-Console, poner la url de la base de datos, que es la siguiente (jdbc:h2:mem:practica4) y usar como usuario y contraseña "admin".

Listado de URL para Postman

Para obtener datos a través de postman se pueden emplear las siguientes URL:

  • Libros:
    1. GET:
      • Mostrar todos los libros: http://localhost:8080/libros
      • Realizar una búsqueda de libros por nombre sustituyendo {name} por el nombre que queremos buscar: http://localhost:8080/libros/busqueda/nombre={name}
      • Buscar un libro determinado por su Id: http://localhost:8080/libro/{id}
      • Realizar una búsqueda de libros por su autor sustituyendo {name} por el nombre que queremos buscar: http://localhost:8080libros/autor={name}
      • Realizar una búsqueda de libros por su editorial sustituyendo {name} por el nombre que queremos buscar: http://localhost:8080libros/editorial={name}
      • Mostrar todos los libros paginados en 3 libros por página, sustituyendo {page} por el número de página que queremos seleccionar: http://localhost:8080/libros/page={page}
      • Realizar una búsqueda de libros por nombre paginados en 3 libros por página, sustituyendo {name} por el nombre del libro y {page} por el número de página que queremos seleccionar: http://localhost:8080/libros/busqueda/nombre={name}/pagina={page}
