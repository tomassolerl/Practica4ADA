Práctica 4

Información

Práctica realizada por Tomás Soler Linares en la que se ha realizado un programa mediante la API Spring Boot que cuenta con una base de datos en la que se encuentran libros, editoriales y autores, y se pueden realizar consultas y transacciones a estos datos mediante el mismo navegador y Postman.

Acceso a la consola

Mediante la consola H2 podemos acceder a las distintas tablas almacenadas en la base de datos y realizar consultas y sentencias SQL.

Para acceder a la consola de la base de datos, hay que acceder a H2-Console, poner la url de la base de datos, que es la siguiente (jdbc:h2:mem:practica4) y usar como usuario y contraseña "admin".

Mostrar datos en el navegador

El acceso al programa desde el propio navegador es mediante este enlace: Libros

Solicitará usuario y contraseña, para acceder hay que introducir el usuario "usuario" y la contraseña "usuario"

Listado de URL para Postman

Para obtener datos a través de postman se pueden emplear las siguientes URL:

  • Libros:
    1. GET:
      • Mostrar todos los libros: http://localhost:8080/libros
      • Realizar una búsqueda de libros por nombre sustituyendo {name} por el nombre que queremos buscar: http://localhost:8080/libros/busqueda/nombre/{name}
      • Buscar un libro determinado por su Id: http://localhost:8080/libro/{id}
      • Realizar una búsqueda de libros por su autor sustituyendo {name} por el nombre que queremos buscar: http://localhost:8080/libros/autor/{name}
      • Realizar una búsqueda de libros por su editorial sustituyendo {name} por el nombre que queremos buscar: http://localhost:8080/libros/editorial/{name}
      • Realizar una búsqueda de libros por su editorial y el nombre del libro, sustituyendo {editorial} por el nombre de la editorial y sustituyendo {name} por el nombre del libro que buscamos: http://localhost:8080/libros/editorial/{editorial}/nombre/{name}
      • Mostrar todos los libros paginados en 3 libros por página, sustituyendo {page} por el número de página que queremos seleccionar: http://localhost:8080/libros/page/{page}
      • Realizar una búsqueda de libros por nombre paginados en 3 libros por página, pasando como parámetros "name" con el valor del nombre y "page" con el número de página: http://localhost:8080/libros/busqueda/?name=Nombre&page=1
    2. POST:
      • Crear un libro pasando como cuerpo del mensaje un libro con todos sus campos menos Id: http://localhost:8080/libro/nuevo
        Ejemplo: { "name": "NuevoLibro", "description": "Descripcion", "editorial": { "id": 4, "name": "Editorial Elegida" }, "autor": { "id": 4, "name": "Autor Escogido" } }
        IMPORTANTE: Los campos autor y editorial deben de existir previamente.
    3. PUT:
      • Modificar o añadir un libro sustituyendo {id} por el Id que queremos añadir/modificar y pasar como cuerpo del mensaje un libro con todos sus campos menos Id: http://localhost:8080/libro/{id}
        Ejemplo: { "name": "NuevoLibro", "description": "Descripcion", "editorial": { "id": 4, "name": "Editorial Elegida" }, "autor": { "id": 4, "name": "Autor Escogido" } }
        IMPORTANTE: Los campos autor y editorial deben de existir previamente.
    4. DELETE:
      • Borrar un libro sustituyendo {id} por el Id de este: http://localhost:8080/libro/{id}
  • Editoriales:
    1. GET:
      • Mostrar todas las editoriales: http://localhost:8080/editoriales
      • Realizar una búsqueda de editoriales por nombre sustituyendo {name} por el nombre que queremos buscar: http://localhost:8080/editorial/busqueda/nombre/{name}
      • Buscar una editorial determinada por su Id: http://localhost:8080/editorial/{id}
      • Realizar una búsqueda de libros por su editorial sustituyendo {name} por el nombre que queremos buscar: http://localhost:8080/editorial/nombre/{name}/libros
      • Mostrar todas las editoriales paginadas en 3 editoriales por página, sustituyendo {page} por el número de página que queremos seleccionar: http://localhost:8080/editoriales/pagina/{page}
      • Realizar una búsqueda de editoriales por nombre paginadas en 3 editoriales por página, pasando como parámetros "name" con el valor del nombre y "page" con el número de página: http://localhost:8080/editoriales/busqueda/?name=Nombre&page=1
    2. POST:
      • Crear una editorial pasando como cuerpo del mensaje una editorial con todos sus campos menos Id: http://localhost:8080/editorial/nuevo
        Ejemplo: { "name": "Editorial Elegida" }
    3. PUT:
      • Modificar o añadir una editorial sustituyendo {id} por el Id que queremos añadir/modificar y pasar como cuerpo del mensaje un libro con todos sus campos menos Id: http://localhost:8080/libro/{id}
        Ejemplo: { "name": "Editorial Elegida" }
    4. DELETE:
      • Borrar una editorial sustituyendo {id} por el Id de este: http://localhost:8080/libro/{id}
  • Autores:
    1. GET:
      • Mostrar todos los autores: http://localhost:8080/autores
      • Realizar una búsqueda de autores por nombre sustituyendo {name} por el nombre que queremos buscar: http://localhost:8080/autor/busqueda/nombre/{name}
      • Buscar un autor determinado por su Id: http://localhost:8080/autor/{id}
      • Realizar una búsqueda de libros por su autor sustituyendo {name} por el nombre que queremos buscar: http://localhost:8080/autor/nombre/{name}/libros
      • Mostrar todos los autores paginados en 3 autores por página, sustituyendo {page} por el número de página que queremos seleccionar: http://localhost:8080/autores/pagina/{page}
      • Realizar una búsqueda de autores por nombre paginados en 3 autores por página, pasando como parámetros "name" con el valor del nombre y "page" con el número de página: http://localhost:8080/autores/busqueda/?name=Nombre&page=1
    2. POST:
      • Crear un autor pasando como cuerpo del mensaje un autor con todos sus campos menos Id: http://localhost:8080/autor/nuevo
        Ejemplo: { "name": "Autor Escogido" }
    3. PUT:
      • Modificar o añadir un autor sustituyendo {id} por el Id que queremos añadir/modificar y pasar como cuerpo del mensaje un libro con todos sus campos menos Id: http://localhost:8080/autor/{id}
        Ejemplo: { "name": "Autor Escogido" }
    4. DELETE:
      • Borrar un autor sustituyendo {id} por el Id de este: http://localhost:8080/autor/{id}
